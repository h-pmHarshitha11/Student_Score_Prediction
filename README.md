# 🎓 Student Score Prediction

## 📌 Project Overview
This project predicts student grades based on study hours using Machine Learning.

The goal is to help identify students who may need extra academic support before exams.

---

## 🧠 Model Used
- Linear Regression
- Supervised Learning
- Built using Scikit-learn

---

## 📊 Dataset
- Source: Kaggle
- Features:
  - study_hours
  - grade

---

## 📈 Model Performance
- R² Score: 0.679

---

## 🔮 Sample Prediction
Input:
Study Hours = 4

Output:
Predicted Grade ≈ 59.91

- Python
- Pandas
