# 🎓 Student Score Prediction

## 📌 Project Description
This project predicts student grades based on the number of study hours using Machine Learning.

The goal is to help identify students who may need extra support before exams.

---

## 🧠 Model Used
- Linear Regression (Supervised Learning)
- Built using Scikit-learn

---

## 📊 Dataset
- Source: Kaggle
- Features:
  - study_hours
  - grade

---

## 📈 Model Performance
- R² Score: 0.679

---

## 🔮 Sample Prediction
Input:
Study Hours = 4

Output:
Predicted Grade ≈ 59.91

---

## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

---

## ▶️ How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the file:
   python student_score_prediction.py
